# Vaultline — Downloads

Private, end-to-end-encrypted group messenger.

**[⬇️ Download the Android APK](https://raw.githubusercontent.com/GUYWGUY/vaultline-releases/main/vaultline.apk)** (arm64, ~71MB)

**Access is limited to approved accounts.** Sign-in is refused for addresses that
are not on the allow list — please contact the maintainer if you need access.

The app checks this repo's `version.json` and offers updates automatically.
No source code lives here — this repository only hosts release artifacts.
